Washington State Ferry Alexa Skill

Initial Proposal

There are many scattered resources available to help passengers deal with the ferries, for example the Washington State Department of Transportation provides an online tracking tool showing where the ferries are in real time. This tool is very convenient when used right but from personal experience I have found it hard to use in a time crunch. Other pieces of data like scheduled sports games and big concerts are out there and are easily accessible and could be utilized to predict an influx of passengers. For my project I want to create an Amazon Alexa skill that organizes various data sources from around the internet and provides the user with careful predictions of ferry departure times that transcend the mere scheduled departure times. This skill will tell the user when the best time to leave for the ferry is and save time normally wasted waiting at the terminal for a delayed boat.
