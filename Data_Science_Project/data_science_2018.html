Willamette University
 
This is the final project that my group did in the Spring of 2018 for CS 429: Introduction to Data Sciene at Willamette University
The project that we chose to do was an old Kaggle challenge. We did not do much image classification during the class so it was up to us to learn about it and implement something. With that said a lot of the code I have posted is taken from other sources. I spent a lot of time making it work for our purposes but I do my best to cite my sources in each of the Jupyter Notebook files. I used Stack Exchange a lot and did not cite everything so big thanks to all the people who contribute to that.
